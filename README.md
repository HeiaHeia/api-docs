api-docs
--------


```
$ git clone https://github.com/HeiaHeia/api-docs.git
$ cd api-docs
# install ruby (apt-get/yum install ruby)
$ gem install bundler
$ bundle install
$ ruby app.rb
```

Visit http://localhost:4567/ in browser.
